# HR Analytics Dashboard using SQL & Power BI

## Project Overview
This project analyzes HR data to understand employee attrition, workforce demographics, salary trends, and departmental performance. SQL was used for data cleaning and analysis, while Power BI was used to build an interactive dashboard.

## Tools Used
- SQL
- Power BI
- Microsoft Excel

## Features
- Data Cleaning
- Exploratory Data Analysis (SQL)
- KPI Cards
- DAX Measures
- Interactive Dashboard
- Slicers and Filters

## Key KPIs
- Total Employees
- Attrition Count
- Attrition Rate
- Average Salary
- Average Age

## Dashboard Visuals
- Attrition by Department
- Attrition by Job Role
- Attrition by Gender
- Attrition by Age Group
- Overtime vs Attrition
- Average Salary by Department
